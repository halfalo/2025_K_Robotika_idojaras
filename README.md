# Robotika beadandó

## A projekt leírása:
### A Projekt célja egy olyan időjárás figyelő állomás létrehozása amely képes páratartalmat, hőmérsékletet és mágnesességet mérni, illetve a szélerősséget egy motorral szimulálja, egy bizonyos erősség felett kigyullad egy led, illetve az adatokat egy kijlezőn jeleníti meg. Az adatok között gombnyomással lehet változtatni.
